# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2026-05-11

### Added

- `templates/AI_WORKFLOW.md` — AI development workflow guide scaffolded into every new project, covering assistant role, core philosophy, architecture rules, milestone workflow, planning rules, implementation rules, development preferences, and response style.

## [0.1.0] - 2026-05-10

### Added

- Initial release: `npx create-jibeos <project-name>` scaffolds README, ROADMAP, MILESTONES, ARCHITECTURE, and CURRENT_STATE markdown files with `{{PROJECT_NAME}}` interpolation.
